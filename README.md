#  Telegram weather bot 

## Installation

### enter api keys to 'config.json'
```json
{
    "telegram_bot_token": "<SECRET_KEY>",
    "owm_token": "<SECRET_KEY>"
}
```
### build & start
```bash
git clone https://github.com/a1excoder/tgweather
cd tgweather/
go build
./weather_bot
```
